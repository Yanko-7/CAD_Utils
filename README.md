format_transform 文件夹下为两个格式转换工具

 - acis_add_gme.py 用于在有gme_前缀函数的文件中生成不带gme_前缀函数的宏控制

 - gme2acis.py     将含有宏控制的或者有gme_前缀相关的全部删除，生成仅acis版本的文件,若要转换成独立版本，可以使用该脚本

gen_uv_img  用于生成模型的UV参数域svg图
